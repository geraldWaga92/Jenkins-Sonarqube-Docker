# DevOps Test Project

A simple CI/CD test application to test the code quality and find potential bugs on our application

this application used technology such as Jenkins, Docker, AWS and SonarQube to test our code

technology used:
 * html, css
 * Jenkins
 * Docker
 * AWS
 * SonarQube
